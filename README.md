# Tomato Leaf Disease Detection using CNN

## 📌 Project Overview
This project is a web-based application that uses Convolutional Neural Networks (CNN) to detect diseases in tomato leaves. Users can upload an image of a tomato leaf, and the system predicts the disease along with accuracy and recommended solutions.

## 🚀 Features
- Upload tomato leaf images
- Detect disease using CNN model
- Display prediction accuracy
- Provide solution for detected disease
- Maintain history of predictions (accuracy table)
- Simple and user-friendly web interface

## 🛠️ Technologies Used
- Python
- Flask (Backend)
- TensorFlow / Keras (Deep Learning)
- HTML, CSS, JavaScript (Frontend)
- SQLite (Database)
